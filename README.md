# Portfolio React - João Washington

Portfólio pessoal desenvolvido com React + Vite.

## Tecnologias

- React 18
- Vite
- CSS3

## Como executar

```bash
# Instalar dependências
npm install

# Executar em modo desenvolvimento
npm run dev

# Build para produção
npm run build

# Preview do build
npm run preview
```

## Estrutura do Projeto

```
src/
├── components/       # Componentes React

│   ├── Skills.jsx

│   └── helpers.js

├── main.jsx         # Entry point

```
